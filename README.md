## Installation

Clone this repo.

```
git clone https://github.com/GZHU-DVL/ADT-SSL.git
```

Prerequisites

- Python=3.8
- pillow
- matplotlib
- pandas
- kornia==0.5.0
- scipy
- scikit-learn
- Pytorch>=1.6.0,<=1.9.0
- Torchvision>=0.7.0,<=0.10.0
- Cudatookit=10.2
